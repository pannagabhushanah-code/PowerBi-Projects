# HR Attrition Analysis – Power BI

## Overview
This Power BI project analyzes employee attrition to identify trends across departments, job roles, gender, and years at company.

## Dashboard Features
- KPI cards: Total Employees, Employee Exits, Exit Rate, Active Employees
- Interactive slicers for Department, Job Role, and Gender
- Attrition analysis by:
  - Department
  - Job Role
  - Gender (Exit %)
  - Years at Company

## Tools & Skills
- Power BI Desktop
- DAX Measures
- Data Modeling
- Data Cleaning

## Key Insights
- Highest attrition observed in Research & Development.
- Employees with fewer years at company show higher exit rates.
- Exit rate differs across job roles and gender.
